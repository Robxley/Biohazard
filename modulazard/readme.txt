Main directory of modulazard project

Requirements:
	- c++17 (with asynchronism/multithreading, pthread on unix system)
	- opencv 

Windows installation:
	- for opencv, use vcpkg install command (see https://github.com/microsoft/vcpkg) as :
		.\vcpkg.exe install opencv:x64-windows	
		
Unix installation:
	- for opencv, use the package manager of your UNIX distribution
